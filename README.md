## Introduce

소프트웨어와 실제 산업 현장이 연결되는 지점을 만드는 개발자입니다.  
AI, 로봇, 설비 데이터, 대시보드를 활용해 현장에서 바로 쓰일 수 있는 자동화 시스템을 구현하는 데 관심이 있습니다.

## Focus

- AI 기반 농장 운영 자동화와 RAG 시스템
- ROS2, SLAM, YOLO 기반 자율주행 정찰 로봇
- OPC UA, Modbus TCP, Qt 기반 스마트팩토리 MES

## Featured Projects

| Project | What it does | Stack |
|---|---|---|
| [All-in-One Fresh Farm](https://github.com/frogjun12-jpg/all-in-one-fresh-farm) | 사과 수확, 선별, 재고, 시세 예측, 판매 등록, 쇼핑몰 주문 관리를 연결한 AI 농장 운영 플랫폼 | Python, FastAPI, Streamlit, MariaDB, RAG, OpenAI/Ollama |
| [RADAR](https://github.com/frogjun12-jpg/RADAR) | TurtleBot3 기반 자율주행 정찰 로봇. 실시간 SLAM과 YOLO 탐지를 통합해 위협 위치를 지도에 표시 | ROS2 Humble, Python, YOLO, TurtleBot3, Qt HUD |
| [Smart_Factory_MES](https://github.com/frogjun12-jpg/Smart_Factory_MES) | OPC UA와 Modbus TCP로 분산 공정 데이터를 통합하고 설비 제어와 생산 이력을 관리하는 MES 프로젝트 | C, C++, Qt 6, MariaDB, OPC UA, Modbus TCP |

## Project Direction

농장 데이터, 로봇, 산업 설비, 대시보드, AI 도우미를 연결해 운영자가 더 빠르고 정확하게 판단할 수 있는 시스템을 만드는 방향으로 프로젝트를 확장하고 있습니다.
